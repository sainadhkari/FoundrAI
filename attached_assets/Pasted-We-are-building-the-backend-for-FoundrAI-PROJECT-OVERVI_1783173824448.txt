We are building the backend for FoundrAI.

PROJECT OVERVIEW:
FoundrAI is an AI Executive War Room for startup founders.

Users submit startup ideas and multiple AI agents analyze:

* market opportunity
* competition
* financial risk
* growth potential
* execution difficulty

Final verdict:

* Proceed
* Pivot
* Reject

CURRENT BACKEND STATUS:
A FastAPI backend already exists with:

* main.py
* app/routes/analyze.py
* app/models/analyze.py
* app/services/analysis_service.py

The current analyze endpoint returns mock data.

Current flow:
Frontend → POST /analyze → analyze_startup() → mock response
